# Backlog

Ideas and future work that hasn't started yet.

## Purpose

Contains:
- Feature ideas
- Bug reports (not yet investigated)
- Technical debt items
- Future enhancements

## Workflow

1. Add idea to `ideas.md` or create specific file
2. When starting work, move to `docs/active/`
3. After completion, move to `docs/completed/YYYY-MM/`

## Structure

```
docs/backlog/
├── README.md    ← This file
├── ideas.md     ← Broad ideas
└── FEATURE-X.md ← Specific feature writeups
```

## Notes

- Not for active work - use `docs/active/`
- Review periodically to prioritize
- Clean up items that are no longer relevant
